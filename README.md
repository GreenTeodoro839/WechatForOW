## 🛠️ 原理
本应用监听系统的通知栏：
1. 当收到微信通知时，检测指定手表的蓝牙连接状态。
2. 如果检测到蓝牙未连接（或系统屏蔽了通道），应用会**重新发送**一条相同内容的通知。
3. 这条新通知会通过正常的系统通道同步到手表。
4. 新通知默认为静默（无声无震动），并在数秒后自动清除，不打扰手机使用。

## 🚀 使用方法
1. 下载并安装 APK。
2. 输入手表的蓝牙 MAC 地址（在手表设置-关于-蓝牙中查看，格式如 `AA:BB:CC:11:22:33`）。
3. 设置延迟销毁时间（建议 3-5 秒）。
4. 点击“保存配置”，并按提示**授予通知权限**。
5. **重要**：在系统设置中，开启本应用的“无障碍服务”以防杀后台。
6. 允许自启动。

## 📦 编译
使用 Android Studio 打开本项目，通过 Gradle 构建即可。
SDK 要求：Android 13.0 (API 33) 及以上。
